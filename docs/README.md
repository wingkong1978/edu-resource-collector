## 📝 资源文件格式

每个资源用一个 Markdown 文件描述：

```markdown
# 资源标题

- **类型**: 视频/文档/试卷/课件
- **学科**: 数学
- **年级**: 高中
- **来源**: B站/知乎/学科网
- **链接**: https://...
- **描述**: 资源内容描述
- **标签**: #函数 #高考 #复习

## 内容摘要

资源的主要内容...

## 适用场景

- 高考复习
- 专题突破
- 基础巩固
```

## 🚀 使用方法

### 1. 手动添加资源

直接在对应学科目录下创建 Markdown 文件：

```bash
cd resources/高中/数学/
vim "001-函数专题讲解.md"
```

### 2. 使用采集脚本

```bash
python collect.py --subject 数学 --grade 高中 --keyword "函数"
```

### 3. 生成汇总索引

```bash
python collect.py --index
```
