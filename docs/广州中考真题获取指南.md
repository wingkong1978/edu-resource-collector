# 广州中考真题获取指南（通用）

📚 本文档汇总广州中考各科真题的通用获取渠道和资源网站

## 🏛️ 一、官方渠道（最权威）

### 1. 广州市招生考试委员会办公室
- **网址**: http://gzzk.gz.gov.cn/
- **内容**: 中考真题、答案、评分标准
- **获取方式**: 每年7月左右公布当年中考试题
- **链接**: http://gzzk.gz.gov.cn/zkzz/zkzt/

### 2. 广州市教育局官网
- **网址**: http://jyj.gz.gov.cn/
- **搜索关键词**: "中考XX试题"（XX为科目名）
- **链接**: http://jyj.gz.gov.cn/zwgk/zdly/jy/zcwj/zcjd/

### 3. 广东省教育考试院
- **网址**: https://eea.gd.gov.cn/
- **内容**: 省统考相关文件、考试说明

## ⚠️ 八、注意事项

1. **真题权威性** - 以广州市招考办公布的为准
2. **答案核对** - 不同来源的答案可能有差异，以官方答案为准
3. **时效性** - 中考政策每年可能有微调，关注最新通知
4. **合理使用** - 真题是宝贵的复习资源，建议限时完成
5. **防范诈骗** - 警惕"花钱买真题""内部资料"等骗局
