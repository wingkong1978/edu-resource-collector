## 🔍 搜索技巧

### 在百度/Google搜索资源
```
关键词 + 文件类型
示例:
- 高中数学函数专题 filetype:pdf
- 高考英语词汇 filetype:doc
- 初中物理实验 video
```

### 在B站搜索课程
```
搜索格式: 学科 + 年级 + 专题
示例:
- 高中数学函数
- 初中物理力学
- 高考英语写作
```

## ⚠️ 使用建议

1. **优先使用官方资源** - 如国家中小学智慧教育平台、人教社官网
2. **注意版权问题** - 下载资源仅供个人学习使用
3. **筛选优质内容** - 视频平台注意选择高质量UP主/讲师
4. **合理利用免费资源** - 很多网站有免费部分，善用即可
5. **保护个人信息** - 注册时注意隐私保护
