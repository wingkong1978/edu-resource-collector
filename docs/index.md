## 🚀 快速开始

### 添加新资源

```bash
python collect.py --add
```

### 生成索引

```bash
python collect.py --index
```

## 📝 资源文件格式

每个资源使用 Markdown 文件记录：

```markdown
# 资源标题

- **类型**: 视频/文档/试卷
- **学科**: 数学
- **年级**: 初中
- **来源**: B站
- **链接**: https://...
- **描述**: 资源内容描述
- **标签**: #函数 #中考

## 内容摘要

资源的主要内容...

## 适用场景

- 高考复习
- 专题突破
```
