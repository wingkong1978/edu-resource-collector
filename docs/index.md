## 🎯 广州中考特色

### 考试科目与分值

| 科目 | 分值 | 考试形式 | 备注 |
|------|------|----------|------|
| 语文 | 120分 | 闭卷 | - |
| 数学 | 120分 | 闭卷 | - |
| 英语 | 120分 | 闭卷+听说 | 笔试90+听说30 |
| 道法 | 90分 | 闭卷 | 与历史合卷 |
| 历史 | 90分 | 闭卷 | 与道法合卷 |
| 物理 | 100分 | 闭卷+实验 | 笔试90+实验10 |
| 化学 | 100分 | 闭卷+实验 | 笔试90+实验10 |
| 体育 | 70分 | 现场测试 | - |

**总分**: 810分

### 重要时间节点

| 时间 | 事项 |
|------|------|
| 3月 | 中考报名 |
| 4月 | 体育考试 |
| 5月 | 英语听说考试、理化实验操作考试 |
| 6月 | 中考文化课考试 |
| 7月 | 成绩查询、志愿填报、录取 |

## 🚀 快速开始

### 添加新资源

```bash
python collect.py --add
```

### 生成本地索引

```bash
python collect.py --index
```

## 📝 资源文件格式

每个资源使用 Markdown 文件记录：

```markdown
# 资源标题

- **类型**: 视频/文档/试卷
- **学科**: 数学
- **年级**: 初中
- **来源**: B站
- **链接**: https://...
- **描述**: 资源内容描述
- **标签**: #函数 #中考

## 内容摘要

资源的主要内容...

## 适用场景

- 中考复习
- 专题突破
```
