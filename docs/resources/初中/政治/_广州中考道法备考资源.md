## 📊 广州中考道法考试说明

### 考试形式与分值
- **考试形式**: **开卷笔试**（可带教材和资料）
- **满分**: 90分（与历史合卷，各90分，共180分）
- **考试时间**: 120分钟（与历史合卷）
- **题型分布**:
  - 选择题: 约40分
  - 主观题: 约50分

### 考试内容结构

| 内容领域 | 分值占比 | 主要内容 |
|----------|----------|----------|
| 道德 | 约30% | 个人品德、家庭美德、社会公德 |
| 法律 | 约35% | 宪法、民法、刑法、未成年人保护法等 |
| 国情 | 约35% | 中国特色社会主义、改革开放、文化建设等 |

### 教材版本
- **部编版**《道德与法治》（人民教育出版社）
- **六册教材**: 
  - 七年级上、下册
  - 八年级上、下册
  - 九年级上、下册

### 开卷考试特点

**可以携带**:
- 六册道德与法治教科书
- 自己做的笔记（不能粘贴他人资料）
- 手写整理的资料

**不可以携带**:
- 复印资料
- 打印资料
- 电子设备
- 夹带纸条

**开卷考试策略**:
- 熟悉教材结构和目录
- 提前准备好速查笔记
- 注重理解和分析能力
- 答题要有条理、有深度

## 🎯 主观题答题技巧

### 辨析题答题模板

**格式**:
```
1. 判断：（正确/错误/片面）
2. 理由：
   - 正确的部分：...
   - 错误的部分：...
   - 正确的观点应该是：...
3. 总结：...
```

### 分析说明题答题模板

**"是什么-为什么-怎么做"模式**:
```
1. 是什么：材料说明了什么现象/问题
2. 为什么：分析原因/意义/影响
3. 怎么做：提出建议/措施
```

**知识点+材料分析模式**:
```
1. 找出材料中的关键词
2. 联系教材知识点
3. 知识点+材料分析
4. 总结升华
```

### 常见答题角度

**个人层面**
- 树立正确观念
- 提高道德修养
- 学习法律知识
- 增强责任意识

**社会层面**
- 加强宣传教育
- 营造良好氛围
- 完善制度机制

**国家层面**
- 坚持依法治国
- 加强法治建设
- 完善法律体系

## 📝 开卷考试速查笔记制作

### 速查笔记内容

**目录索引**
- 按教材目录制作
- 按知识点分类
- 按题型分类

**重点知识**
- 核心概念
- 重要观点
- 法律依据
- 政策措施

**答题模板**
- 辨析题模板
- 分析题模板
- 探究题模板

### 制作建议

1. **手写整理** - 加深记忆，符合开卷要求
2. **颜色标记** - 不同颜色标注重点
3. **索引标签** - 方便快速查找
4. **简洁明了** - 不写长段落，写关键词
5. **提前准备** - 不要等到考前才整理

### 速查笔记示例格式

```
【宪法专题】
- 宪法地位：根本法、最高法律效力
- 宪法原则：人民主权、基本人权、法治、民主集中制
- 公民基本权利：...
- 公民基本义务：...

【法律题型】答题角度：
1. 法律规定
2. 权利义务
3. 法律责任
4. 青少年启示
```
